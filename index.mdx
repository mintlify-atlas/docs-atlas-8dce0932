KilomeTracker is a vehicle management API designed for individual drivers and small fleets. It gives you a complete picture of your vehicles: every kilometer driven, every fuel stop, every service visit, and every associated expense — all in one place.

The API exposes over 40 endpoints organized by resource, with JWT-based auth, role-based permissions, and pagination on every list endpoint.

## How it works

<Steps>
  <Step title="Register and get a token">
    Call `POST /api/auth/register` to create an account. You'll receive a JWT token to use on subsequent requests.
  </Step>
  <Step title="Add a vehicle">
    Call `POST /api/vehicles` with an alias, make, model, and initial odometer reading. The API tracks the total from there.
  </Step>
  <Step title="Log activity">
    Record trips, refuels, maintenance visits, and expenses against your vehicle using its alias.
  </Step>
  <Step title="Analyze and act">
    Query `/api/vehicles/:alias/stats` or `/api/dashboard` for aggregated cost-of-ownership data and efficiency metrics.
  </Step>
</Steps>

## Base URL

All endpoints are relative to your deployment base URL:

```
https://your-deployment.railway.app
```

All requests and responses use `Content-Type: application/json`.

<Note>
  The currency used throughout the system is the **Guatemalan Quetzal (Q)**. Monetary fields like `cantidadGastada`, `costo`, and `monto` are denominated in GTQ.
</Note>

title: 'Maintenance'
description: 'Log service records and get alerts for upcoming maintenance due by date or mileage.'

This page covers six endpoints for creating and managing vehicle maintenance records. GET operations require authentication; write and delete operations additionally require the `write` or `admin` role.

---

## List maintenance records

`GET /api/maintenance`

Returns all maintenance records owned by the authenticated user. Supports filtering by vehicle, service type, and date range.

**Auth required:** Yes — `Authorization: Bearer <token>`

### Query parameters

<ParamField query="vehicleAlias" type="string">
  Filter records to a specific vehicle by its alias (case-insensitive).
</ParamField>

<ParamField query="tipo" type="string">
  Filter by maintenance type. One of: `Cambio de aceite`, `Rotación de llantas`, `Frenos`, `Inspección`, `Reparación`, `Batería`, `Filtros`, `Transmisión`, `Suspensión`, `Alineación`, `Otro`.
</ParamField>

<ParamField query="startDate" type="string">
  Earliest service date to include, in `YYYY-MM-DD` format.
</ParamField>

<ParamField query="endDate" type="string">
  Latest service date to include, in `YYYY-MM-DD` format.
</ParamField>

### Example

```bash cURL
curl --request GET \
  --url 'https://api.kilometracker.app/api/maintenance?vehicleAlias=CAR1&tipo=Cambio%20de%20aceite&startDate=2024-01-01&endDate=2024-12-31' \
  --header 'Authorization: Bearer <token>'
```

## Get upcoming maintenance

`GET /api/maintenance/upcoming`

Returns maintenance records whose next scheduled service falls within the next 30 days or within 500 km of the vehicle's current odometer reading.

**Auth required:** Yes — `Authorization: Bearer <token>`

### Example

```bash cURL
curl --request GET \
  --url https://api.kilometracker.app/api/maintenance/upcoming \
  --header 'Authorization: Bearer <token>'
```

```json Response
{
  "success": true,
  "count": 1,
  "data": [

}
```

## Create a maintenance record

`POST /api/maintenance`

Creates a new maintenance record for a vehicle.

**Auth required:** Yes — `write` or `admin` role

<Tip>
  Populate `proximoServicioFecha` and `proximoServicioKm` when creating a record to enable future alerts from the `/upcoming` endpoint.
</Tip>

### Request body

<ParamField body="vehicleAlias" type="string" required>
  Alias of the vehicle this record belongs to.
</ParamField>

<ParamField body="tipo" type="string" required>
  Type of maintenance performed. One of: `Cambio de aceite`, `Rotación de llantas`, `Frenos`, `Inspección`, `Reparación`, `Batería`, `Filtros`, `Transmisión`, `Suspensión`, `Alineación`, `Otro`.
</ParamField>

<ParamField body="descripcion" type="string" required>
  Description of the work performed.
</ParamField>

<ParamField body="costo" type="number" required>
  Cost of the service in Q (quetzales). Minimum: `0`.
</ParamField>

<ParamField body="kilometraje" type="number" required>
  Odometer reading at the time of service in km. Minimum: `0`.
</ParamField>

<ParamField body="fecha" type="string">
  Date the service was performed, in ISO 8601 format. Defaults to the current date and time.
</ParamField>

<ParamField body="proveedor" type="string">
  Name of the workshop or service provider.
</ParamField>

<ParamField body="proximoServicioFecha" type="string">
  Date when the next service is due, in ISO 8601 format.
</ParamField>

<ParamField body="proximoServicioKm" type="number">
  Odometer reading at which the next service is due. Minimum: `0`.
</ParamField>

<ParamField body="notas" type="string">
  Additional notes about the service.
</ParamField>

### Example

```bash cURL
curl --request POST \
  --url https://api.kilometracker.app/api/maintenance \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "vehicleAlias": "CAR1",
    "tipo": "Cambio de aceite",
    "descripcion": "Cambio de aceite sintético 5W-30",
    "costo": 350,
    "kilometraje": 15000,
    "proveedor": "Taller Mecánico XYZ",
    "proximoServicioFecha": "2025-05-24",
    "proximoServicioKm": 20000,
    "notas": "Incluye cambio de filtro"
  }'
```


## Update a maintenance record

`PUT /api/maintenance/:id`

Updates one or more fields on an existing maintenance record. Only the fields you include in the request body are changed.

**Auth required:** Yes — `write` or `admin` role

### Request body

<ParamField body="tipo" type="string">
  Updated maintenance type.
</ParamField>

<ParamField body="descripcion" type="string">
  Updated description.
</ParamField>

<ParamField body="costo" type="number">
  Updated cost in Q. Minimum: `0`.
</ParamField>

<ParamField body="fecha" type="string">
  Updated service date in ISO 8601 format.
</ParamField>

<ParamField body="kilometraje" type="number">
  Updated odometer reading. Minimum: `0`.
</ParamField>

<ParamField body="proveedor" type="string">
  Updated service provider name.
</ParamField>

<ParamField body="proximoServicioFecha" type="string">
  Updated next service date in ISO 8601 format.
</ParamField>

<ParamField body="proximoServicioKm" type="number">
  Updated next service odometer reading. Minimum: `0`.
</ParamField>

<ParamField body="notas" type="string">
  Updated notes.
</ParamField>

### Example

```bash cURL
curl --request PUT \
  --url https://api.kilometracker.app/api/maintenance/6642f3a1c3d4e5f6a7b8c9d0 \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "costo": 380,
    "notas": "Incluye cambio de filtro y revisión de frenos"
  }'
```

```json Response
{
  "success": true,
  "data": {
    "_id": "6642f3a1c3d4e5f6a7b8c9d0",
    "vehicleAlias": "CAR1",
    "tipo": "Cambio de aceite",
    "descripcion": "Cambio de aceite sintético 5W-30",
    "costo": 380
  }
}
```

## Delete a maintenance record

`DELETE /api/maintenance/:id`

Permanently deletes a maintenance record. This action cannot be undone.

**Auth required:** Yes — `write` or `admin` role

<Warning>
  This is a permanent delete. The record cannot be recovered after deletion.
</Warning>

### Example

```bash cURL
curl --request DELETE \
  --url https://api.kilometracker.app/api/maintenance/6642f3a1c3d4e5f6a7b8c9d0 \
  --header 'Authorization: Bearer <token>'
```

```json Response
{
  "success": true,
  "message": "Mantenimiento eliminado correctamente"
}
```
